### Funzionamento
Lo script aggiunge al file hosts la maggior parte dei servizi ai rimappandoli per puntare a localhost, in modo che la pagina non venga caricata. 

⚠️ Viene bloccato anche google.com per evitare l'uso di AI Overview. In alternativa si può usare startpage.com come motore di ricerca. L'estensione permette di impostarlo in automatico come motore predefinito. Disinstallando l'estensione, ritornerà il motore di ricerca predefinito precedentemente impostato.

### Per Bloccare
